# Setup App de Administración

Configuración inicial de la aplicación de administración (backoffice) para Vichente Eats.

## ✅ Completado

### Tecnologías implementadas:

- **Next.js 15.3.5** - Framework React con App Router
- **TypeScript** - Tipado estático
- **Emotion** - CSS-in-JS para styled components
- **React 19** - Biblioteca de UI

- ESLint configurado
